# Resume Setup

## Instructions

The glitch download button has been added to your portfolio! To make it functional, you need to:

1. **Create your resume PDF** and save it as `resume.pdf` in the `public/` folder
2. The button will automatically download the file when clicked

### File Location
- Place your resume at: `public/resume.pdf`
- The button will download it as: `Ritvik_Roy_Resume.pdf`

### What the Button Does
- Triggers an automatic download of `resume.pdf` from the public folder
- The downloaded file will be named `Ritvik_Roy_Resume.pdf`
- Features a unique glitching animation effect that matches your portfolio theme:
  - RGB channel separation glitch on hover
  - Scan line animation
  - Glow effect
  - Smooth scale transitions

### Styling Features
- Antireal purple color theme matching your portfolio
- Animated glitch effect that intensifies on hover
- Responsive design (mobile and desktop)
- Mix-blend-screen effect for RGB channels
- Scan lines that animate when hovered
